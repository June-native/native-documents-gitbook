# Native Core

**Native Core** is a high-performance, first-in-first-out (FIFO) central limit order book (CLOB) that is fully on-chain — the sovereign state machine where Native's **pricing logic and order matching are finalized transparently**.

It is built for traders who want ultra high performance with full on-chain verifiability.

* **Fully On-chain Verifiable:** all actions are publicly auditable.
* **High Throughput, Low Latency:** high bandwidth with sub-second finality.
* **Fair Execution:** non-biased FIFO (first-in-first-out) order matching
