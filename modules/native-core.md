# Native Core

**Native Core** is a high-performance central limit order book (CLOB) application chain — the sovereign state machine where Native's **pricing logic and order matching are finalized transparently on-chain**.

It is built for traders and validators who want centralized-exchange-grade performance with full on-chain verifiability.

* **Fully On-chain Verifiable:** all actions are on-chain and publicly auditable.
* **High Throughput, Low Latency:** high TPS with sub-second finality.
* **Direct Access:** accessible with standard trading APIs.
