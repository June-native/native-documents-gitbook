# Native Core

**Native Core** is a high-performance central limit order book (CLOB) application chain — the sovereign state machine where Native's **pricing logic and order matching are finalized on-chain**.

It is built for traders and validators who want centralized-exchange-grade performance with full on-chain verifiability.

* **Fully on-chain verifiable:** all actions are on-chain and publicly auditable.
* **High throughput, low latency:** high TPS with sub-second finality.
* **Spot-first:** designed and optimised for spot.
